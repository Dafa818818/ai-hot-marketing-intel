# AI 热点与营销投放情报

这是一个面向运营、市场、广告投放同事的公开静态看板。主数据源是 AI HOT，扩展信息来自公开 RSS/Atom 来源，不写入公司敏感信息、客户信息、预算或内部策略。

## 自动更新

- GitHub Actions 每天北京时间 09:00 触发一次。
- 脚本会先判断当天是否为周一至周五，且不是中国法定节假日。
- 周末和假期不会改动页面。
- 周一或假期后第一个工作日，会从上次成功更新的 09:00 补到本次 09:00，避免漏掉周末或假期消息。
- 页面只保留最近 5 个成功更新日的归档。

## 版本回溯

- 每次工作日云端成功更新后，GitHub 会生成一次提交记录。
- 同时会自动创建一个版本标签，命名格式为 `ai-intel-YYYY-MM-DD`。
- 如果需要回看某一期，可在 GitHub 的 Tags/标签里按日期查找。
- 页面只展示最近 5 个更新日，但 GitHub 的提交历史和标签会保留更早版本，方便回溯。

## Notion 同步

- 每次页面数据成功更新后，GitHub Actions 会尝试把最新一期同步到 Notion。
- 同步内容包括：每日更新总览、每条热点、当期投放关键词池。
- 同步脚本会先检查 Notion 是否已有同一期，已有则跳过，避免重复写入。
- Notion 授权只通过 GitHub Secrets 保存，不写进公开仓库。
- 如果没有配置 Notion Secrets，网页更新仍会正常进行，只会跳过 Notion 同步。

## 成本

默认方案使用 GitHub Pages、GitHub Actions、AI HOT 公开接口和公开 RSS/Atom 来源，不需要付费 AI API，不需要信用卡。

## 每年维护节假日

国务院公布下一年度放假安排后，更新 `data/holidays-cn-2026.json`，把年份、非工作日和来源链接替换为新年度即可。项目默认按“周一至周五且非法定节假日”更新；周六、周日即使调休上班也不自动更新。
